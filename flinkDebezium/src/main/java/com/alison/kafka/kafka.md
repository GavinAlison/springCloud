# Kafka的管理Java客户端

Kafka的管理Java客户端，支持管理和检查topic、broker、配置和ACL。

## create topic
cmd
```text
bin/kafka-topics.sh --create --bootstrap-server localhost:9092, localhost:9093,localhost:9094  --replication-factor 3 --partitions 30 --topic 811_dml_maxwell

bin/kafka-topics.sh --create --bootstrap-server localhost:9092, localhost:9093,localhost:9094  --replication-factor 3 --partitions 30 --topic 811_dml_maxwell

bin/kafka-topics.sh --create --bootstrap-server localhost:9092, localhost:9093,localhost:9094  --replication-factor 3 --partitions 30 --topic 811_dml_maxwell_compensation

```
